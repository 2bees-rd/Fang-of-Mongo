Fang-of-Mongo by Fiedzia

web based gui for mongodb. See live demo at http://blueone.pl:8001/fangofmongo/

Fang of Mongo - web based user interface for MongoDB build with django and jquery. It will allow you to explore content of mongodb with simple but (hopefully) pleasant user interface. Project goals: - allow users to easily explore mongdb content and any metainformation it may provide - Long term goal is to being able to perform any operation that can be done using mongodb console

Live demo
